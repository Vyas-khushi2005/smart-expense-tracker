# Smart Expense Tracker

## Overview
Smart Expense Tracker is a responsive web application that helps users manage their daily expenses. It allows users to add, view, and organize expense records through a simple and user-friendly interface.

## Features
- Add new expenses
- Display expense records
- Responsive user interface
- Clean and easy-to-use design

## Technologies Used
- HTML5
- CSS3
- JavaScript
